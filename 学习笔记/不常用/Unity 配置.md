## 配置

**设置 vscode 作为默认 IDE**

1. vscode 安装插件：C#，Unity tools
    可选插件：Unity Snippets，C# XML Documentation Comments（似乎其它插件里带了）。
2. Window > PackManager > 安装 visual studio code editor（可能是 visual studio editor；可能会默认装，可以 unlock 然后更新一下）
3. Preferences > External Tools 设置 vscode，然后点一下 regenerate project files。
