## 快捷键

### iTerm2

可以导出导入快捷键。

- 新建标签：command T
- 关闭标签：command W
- 切换全屏：command enter
- 垂直分屏：command d
- 水平分屏：command shift d
- 查看历史命令：command ;
- 查看剪贴板历史：command shift h

- 清除行：ctrl u
- 搜索历史命令：ctrl r
- 清屏：command r，ctrl l
- 行首/行末：ctrl a/e，command left/right（配置为0x01与0x05）
- 删除一个词（当前光标到词开头）：ctrl w（注意不是 command w）

### 软件




### Python

pip3 安装的包在虚拟环境 Work/my_py_venv 下。
通过`source my_py_venv/bin/activate`进入环境，然后可安装包`python3 -m pip install matplotlib`或运行 python。

可能要`pip install xxx --timeout 60`来避免 ks 镜像超时、安装失败。
或者指定镜像：`pip install tensorflow -i https://pypi.tuna.tsinghua.edu.cn/simple`


### venv

> 注意，如果更改虚拟环境（即 activate）的所在目录，需要更新 activate 文件中的目录，然后 source 才有用。
> 可能要重新选择右下角的解释器路径。

```bash
# 在指定路径创建 venv
python3 -m venv path/to/work
source path/to/work/bin/activate
python3 -m pip install xyz

# 退出
deactivate

# 查看当前 venv 安装的库
pip3 list
# 查看已安装特定库的信息
pip3 show <name>
```


### pyenv

可以用 pyenv 安装多个 python 版本，配合 venv。

设置全局 python 版本：`pyenv global 3.12.0`
设置当前目录下 python 版本：`pyenv local 3.6.8`
查看使用的版本：`pyenv version`
列出所有已安装版本：`pyenv versions`
切换版本：`pyenv shell 3.12.0`

mac M1 会因为 arch 是 arm64 安装不了低版本 python，需要：`arch -x86_64 pyenv install 3.8.6`
python 3.6 和 3.7 好像没办法或很难，因此装不了 tf1，只能通过下面的方式用 tf2 模拟 tf1。

```python
import tensorflow.compat.v1 as tf
tf.disable_v2_behavior()
```

或者试试：https://www.zhihu.com/question/489896424/answer/2721373841

### conda

查看当前激活的环境：`conda info --envs`
激活某个环境：`conda activate your_environment_name`
取消激活 conda 环境：`conda deactivate`

`which python3`查看当前的 python 是哪个环境的。
